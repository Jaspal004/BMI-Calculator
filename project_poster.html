AI-Based BMI Estimation From Body Proportions

A computer vision and machine learning system that estimates BMI from a full-body image using pose landmarks, silhouette measurements, and a Random Forest regression model.

Project Goal

The system predicts BMI without manual weighing by converting a full-body image into measurable body-proportion features. It returns a BMI estimate, BMI category, reliability score, previous prediction history, and trend visualization through a simple web interface.

How It Works

1
Image Input User uploads a full-body image or opens the live camera route.
2
Pose Detection MediaPipe detects body landmarks and segmentation mask.
3
Feature Vector Shoulder, hip, height, torso, arm, leg, and ratios are extracted.
4
ML Prediction A Random Forest regressor predicts a continuous BMI value.
5
Result Screen The app displays BMI category, reliability, history, and trend.

BMI Classes

The app classifies predictions into familiar health ranges, with a small margin near class boundaries to reduce overreaction to image-based estimation noise.

Key Takeaways

Image-Based Estimation The project demonstrates how visual body proportions can be transformed into a numeric BMI estimate.
Interpretable Features The model uses measurable landmarks and ratios instead of hidden image embeddings, making the workflow easier to explain.
Usable Interface The Flask app connects upload, prediction, category display, reliability checking, and history tracking.
